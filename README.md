# KeenKeeper

## Short Description

KeenKeeper is a friendship management web app where users can keep track of friends, open individual friend profiles, log interactions like calls, texts, and video calls, and view those activities in a timeline and analytics dashboard.

## Technologies Used

- Next.js
- React
- Tailwind CSS
- DaisyUI
- Recharts
- React Toastify
- Font Awesome

## Key Features

- Dynamic friend list and friend profile pages powered by local JSON data.
- Timeline history that records and displays call, text, and video interactions.
- Analytics dashboard with a pie chart that shows interaction type breakdown.
